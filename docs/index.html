What it does

Three quiet upgrades to a loud platform

SNapp doesn't replace ServiceNow. It sits next to it and handles the things the form was never going to handle for you.

🔗

Extracted Links Panel

Scans every form field for URLs and surfaces them in a Related Links panel with friendly labels — GitHub, Jira, Confluence, and more — instead of buried inside long description text.

✉

Email Contacts Panel

Pulls email addresses out of any field on the page and presents them as one-click mailto: links. No more copy-paste-into-a-new-tab dance.

⌘

Right-Click Context Menu

Copy richly formatted record links, change request metadata, or incident summaries with one click — already laid out for Slack, email, or a ticket comment.

Compatibility

Works where you work

Manifest V3, no runtime dependencies, no telemetry. The extension you install does exactly what the README says — nothing more.

A short note on the rebrand

SNapp ran in production for a while under a different name. It worked — daily users, real value, the kind of feedback that tells you a tool earned its keep.

We pulled it back to relaunch it cleanly under ShadoLogiq Labs: new branding, a fresh public release, and the same focus on local-authority browser tools that respect your time.

No date promises — quality over deadline. The fastest way to know when SNapp ships: watch the snapp-ext repo and you'll see the release the moment it lands.

Watch this space.
